# 策略规则 · 科技强势 + 回调 + 二波（finance-strategy）

> 个人 A 股短线/波段框架，供 `finance-strategy` 生成候选池与信号时引用。

## 1. universe（股票池范围）

- 主板/创业板/科创板 **科技成长**：半导体、算力、光模块、设备、AI 应用等
- 排除：ST、退市风险、日均成交额长期低于门槛（见 `finance-risk` 流动性规则）

## 2. 强势定义（日线）

同时满足 ≥3 项可标「强势」：

1. 近 10 日涨幅位于同概念前 30%
2. 收盘价在 20 日均线之上，且 20 日线向上
3. 近 5 日至少 1 日放量（成交额 > 前 20 日均量 1.5 倍）
4. 板块指数同期偏强（妙想/行业数据佐证）

## 3. 回调买点（第一次介入）

- 强势标的出现 **3～8%** 日线回调（或回踩 10/20 日线不破）
- 回调日缩量（成交额 < 5 日均量 0.8）
- 无重大利空公告（`finance-data` 舆情无「严重」标签）

## 4. 二波（加仓/持有）

- 回调后再次放量突破回调高点，或突破近 10 日高点
- 板块仍处强势梯队；**不追** 涨停封板无法排板价位

## 5. 卖出/减仓（策略侧建议，风控终审）

- 跌破 20 日线且 3 日未收回
- 单日放量长上影 + 跌幅 > 5%
- 板块退潮（龙头集体破位）

## 6. 输出契约

| 产出 | 路径 |
|------|------|
| 每日候选池 | `workers/strategy/pool/YYYY-MM-DD_候选池.md` |
| 盘中预警 | `workers/strategy/signals/盘中预警_YYYY-MM-DD_HHmm.md` |
| 买卖信号 | `workers/strategy/signals/信号_YYYY-MM-DD_HHmm.json` |
| 调仓方案 | `workers/strategy/reports/调仓交易方案_YYYY-MM-DD.md` |

信号 JSON 字段：`code`, `name`, `side`, `reason`, `confidence`, `ref_price`, `strategy_tag`（强势/回调/二波）
